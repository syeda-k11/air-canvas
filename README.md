# Air Canvas using AI

Air Canvas is an AI-powered virtual drawing tool that lets you draw in the air using hand gestures captured through your webcam. It uses real-time hand tracking and computer vision to convert your finger movements into digital strokes on the screenâ€”no touch or stylus needed.

## Features

- Real-time hand gesture recognition
- Draw in the air with your index finger
- Change colors with gesture controls
- Erase drawings with pinch gesture
- Clear canvas with open palm gesture
- Web-based interface with responsive design

## Technologies Used

- Python 3.12.10
- OpenCV
- MediaPipe
- NumPy
- Flask
- HTML/CSS/JavaScript

## Installation

1. enter into directory:
   cd air-canvas-ai

2. Create a Virtual Environment:
   python -m venv venv

3. Activate the Virtual Environment:
   .\venv\Scripts\activate

4. Install Dependencies:
   pip install --upgrade pip
   pip install -r requirements.txt

5. Verify the Installation
   pip list

6. Running the Application
   python app.py

7. Deactivating the Virtual Environment
   deactivate
